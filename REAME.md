# Git Commands

## Check status
git status

## Add a specific file
git add filename.txt

## Add all changed files
git add .

## Commits
git commit -m "commit"

## Upload to remote 
git push origin main 

## Create a new branch 
git branch branch_name

## Switch to branch 
git checkout branch_name 

## List all branches
git branch 

## Delete branch 
git branch -d branch_name

## Roll back to a Commit 
git reset --hard <commit-hash>
